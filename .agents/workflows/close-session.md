---
description: How to close a session properly in QuantLab
---

1. Ensure all code changes are verified by tests.
2. Update `.agents/current-state.md` with final status.
3. Add a summary of work to `.agents/session_log.md`.
4. list any "Next Steps" for the user.
5. Create a `walkthrough.md` artifact if significant features were added.
6. Do not commit or push (per project policy).
